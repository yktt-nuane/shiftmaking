## Prerequisites

- Node.js (version 18.11.11 or higher)
  - Next.js (version 13.0.6 or higher)
  - ESLint
  - Prettier
  - Stylelint

## How to

### Develop

`npm run dev` to start development server.

### Deploy

Every time you push to GitHub, it is automatically deployed to Vercel.
